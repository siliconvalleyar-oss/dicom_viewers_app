# Codebuff Skills — DICOM Visual

Skills documentando patrones y conocimiento adquirido durante el desarrollo de la app **DICOM Visual**.

## Skills Disponibles

| Skill | Descripción |
|---|---|
| `flutter-dicom-setup` | Configuración del proyecto Flutter, dependencias, Android, firma release |
| `orthanc-api-integration` | Integración con servidor Orthanc, endpoints, modelos de datos, autenticación |
| `dicom-image-viewer` | Visor de imágenes DICOM: WW/WL, zoom, galería, metadatos, overlay |
| `flutter-gesture-handling` | Manejo de gestos: modo toggle WW/WL vs Zoom, SafeArea, PageView |

## Uso

Para cargar un skill:
```
@skill flutter-dicom-setup
```
